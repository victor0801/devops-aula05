# Arquitetura

* As funções relacionadas ao gerenciamento das casas do jogo da velha ficarão no modulo **jogovelha.py**.

* O estado de cada casa do jogo sera representada por uma string: "." para casa vazia; "x" para casa ocupada pelo 1o jogador; "o" para casa ocupada pelo 2o jogador

" A função inicializar () retornara ujma lista 3x3, onde cada posição conterá uma string para indicar o estado de uma casa do jogo. A função retornara todas as casa inicialmente vazias.
